 Setting up your website template:
Step 1:
 EDIT FLASH ANIMATION: | Click for more details
Open the "flash.txt" file with Notepad or any text editor and edit "Yoursitetitle.com" with your name or company name. Be sure not to delete the "logo=". Edit the "&logo-faq=Yoursitetitle.com F.A.Q." with your FAQ popup title. Be sure not to delete the "&logo-faq=". This will update the Flash animation on all pages. If you wish to use a graphic logo, there are instructions later in this document.

HOMEPAGE FLASH IMAGE SHOW: | Click for more details
The animation in the index.html is showing 8 .jpg images. You will find these images in the "flash" folder. You can replace "image1.jpg" through "image8.jpg" with any standard .jpg images and the flash will update. Make your images 400 x 325 pixels in size.

Step 2:
 EDIT COPYRIGHT: | Click for more details
Open the "copyright.js" file in Notepad or any text editor and edit "Your Company" with your website title. This will update the footer info on all pages.

Step 3:
 EDIT CONTACT INFO: | Click for more details
Open the "contact.js" file in Notepad or any text editor and change the company name, address, phone and fax numbers and e-mail address (edit the e-mail in 2 places). This will update contact info on the "contact" page. This is setup in a .js file so search engines won't see your e-mail and you'll get less spam.

Step 4:
 EDIT THE CONTACT AND QUOTES FORM:

Click for current info on setting up the template forms.

Click for info on adding more fields to the forms.

Click for changing form sizes or colors.

If you want to use only e-mail on the contact page you can remove the contact form from the "contact.htm" between the comment tags.

Because of the complexity of setting up the quote form, there are notes in the "quotes.htm" to help you cut and paste to add new form fields. The form will work with most contact form scripts however you will need help files or a sample form from the script you intend to use. Most form elements work the same way from script to script, however the hidden values usually vary. You may want to consider editing the entire page in Notepad instead of a wysiwyg editor.

Step 5:
 GALLERY SETUP: | More details | Thumbnail Help
The best way to setup the pictures in the 3 gallery.htm pages is not to actually edit the HTML pages and change the pictures on each individual page. The fastest way is to simply replace the pictures in the "gallery" folder with your own by overwriting the files that are in there now and using the same generic names.

In the "gallery" folder, replace the 12 "Fgallery" pictures with full size photos. Replace the 12 "gallery" pictures with your product thumbnails. The thumbnails are now 130 x 87 pixels and the full sized images are 500 x 333. You can make them any size that you prefer. There is info on the other pictures at this site later in this document. Also see "options" for details on the slideshow/tour page.

IMAGE VIEWER OPTIONS: | Click for more details
You can change some options for the gallery image viewer page (image-viewer.htm) by editing the "pop-closeup.js" in Notepad. Options include new browser window, or popup for the viewer.

Step 6:
 CART SETUP: | 2CO Setup | Paypal Setup
  1. If you are using the gallery as a shopping cart, move the 3 "NOCARTgallery?.htm" pages into the template "extras" folder.
  2. Click here for the steps to setup the 2CO forms.
  3. Click here for the steps to setup the Paypal forms.
  4. Once you have setup the forms for Paypal and 2CO edit 2 variables in the sidebar.js for the view cart buttons. Find "var paypal_email" and replace "paypal@your-web-domain.com" with your Paypal e-mail. Find the var "TWOCO_sellerID" and edit "999999" with your 2CO seller ID. The ID can be found by logging into your 2CO account.
NOTE: If you want to remove either the Paypal or the 2CO cart: In each gallery HTML page delete the code between the "2cO FORM" or "PAYPAL FORM" notes for each item. Then turn off the view cart button in your sidebar.js.

Step 7:
 EDIT FAQ PAGES: | Click for more details
See your FAQ page for complete instructions.

Step 8:
 SETUP THE SITE SEARCH:
Open the "search.js" and edit "allwebcodesign.com" with your website domain. To turn off the site search, open the "search.js" and edit the show search to "no".

If you require a more custom search with no ads, you can use the "OPTIONAL-Burgdorf-search.js" in your "extras" folder. Click here for help setting up the Burgdorf search.

To use a Google only search, move the "search.js" into your "extras" folder. Move the "OPTIONAL-GOOGLE-search.js" into the main template folder and rename that to "search.js".

Step 9:
 EDIT PAGES: | Software choices | Notepad Editing
Edit the text on all the HTML pages with your website information. You can use EW, Frontpage, Dreamweaver, GoLive, any wysiwyg HTML editor or a text editor like Notepad. Take care not to delete any of the HTML tags that surround the text.

Step 10:
 EDIT TITLES, DESCRIPTIONS AND KEYWORDS: | Click for more details
Open each of the HTML pages and change the titles (at the very top of the page) and also the description and the keywords with your product or service info. You can edit one page and copy and paste to the other pages. Be careful to only change what's inside the "" so you don't disturb the html code. Page titles are important so your site will get a good listing at Google or MSN and other search engines. Click for optimization help.

Step 11:
 UPLOAD: | Click for more details
Upload the template files on to your hosting company's server using ftp software or Windows Explorer or your control panel. Be sure to upload the "picts" and "flash" and and "gallery" folders and all files in these folders.



Options:
USING A GRAPHIC LOGO:
You can replace the Flash logo with a graphic logo. To do this, delete or rename the "header.js", then rename the "graphic_logo_header.js" to "header.js". In the picts folder you will find a graphic called "logo.jpg". Edit this graphic with your logo. It's best to keep the logo.jpg graphic around the same size that it is now.

paypal.htm PAYPAL PAYMENT PAGE:
This template includes a Paypal payment page called "paypal.htm". Edit the "paypal@your-web-domain.com" in this page with your Paypal e-mail and you can use this page to accept custom payments by sending clients a link to this page and having them fill in any amount on the payment page.

THE ALL PRODUCTS PAGE (site_map.htm): | Click for details
The site map included in this template was created so all your pages will be listed at major search engines. Be sure to add links to any new pages on this page. Leave a text link to the "site_map.htm" in the homepage (index.html). A link from the "menu.js" may not be seen by some search engines, so a link is already included on the homepage.

PAGE HEIGHTS:
So all your pages will maintain a standard height a "pageheight" class can be found in the "coolstyle.css". Edit the height for this class to any height you need.

THE "extras" FOLDER:
The extras folder has some source PSD Photoshop images you can use. You do not need to upload this folder to your hosting area.

THE NOCARTgallery1.htm PAGES:
3 plain gallery pages are included in case you do not require a shopping cart or just want to add some image gallery pages. To use these pages instead of the shopping cart, move the 3 "gallery?.htm" pages into the template "extras" folder. Rename the 3 "NOCARTgallery?.htm" pages to "gallery1.htm", "gallery2.htm" and "gallery3.htm".

THE robots.txt FILE:
The template robots.txt is setup to keep search engines out of some folders like the "cgi-bin". You can upload this file to your hosting area but it is not required.

SETTING UP THE PRODUCT TOUR (SLIDESHOW) PICTURES: | Click for more details
The slideshow.htm is currently setup to display all pictures in the gallery folder. If you would like to add or remove pictures from the "product tour" slideshow, open the slideshow.htm in Notepad and look for the "EDIT THIS AREA" note. Copy and paste to add new pictures to the list, or delete lines to remove pictures. You can also set the AutoPlay speed. Look for "var rotate_delay = 3000". To remove the "Products Tour" button in the image closeups edit the "image-viewer.htm".

EDITING FONTS OR COLORS: | Click for font details | Click for color details
You can change your font colors and sizes by editing the "coolstyle.css" with a text editor. Edit the "TABLE" px size to change the copy on all pages. Change the "title" px size to change all the title sizes.

In this template many area colors are defined in the "coolstyle.css" file. Look for the /* PAGE COLORS */ note for the main template colors. Click here for more color help.

CHANGING LINKS / ADDING TO THE MENU: | Click for more details
This template includes 2 menus you can add and remove links in.
1. menu.js (left menu)
2. menu_horz.js (page top menu)
You can change the names of the links like "about" or "contact" to other page names by editing any one of the menu.js files mentioned above. Only change the one instance of the name and not the name with the ".htm" after it. Whatever you change in any of the .js menu files will change every page on the website. You can add more pages by copying one link in any menu .js file and pasting it right below the itself. Be sure to back up your files before you edit them. There are notes in the ".js files for adding links. If you add pages be sure to add any new links to the "site_map.htm".

ADDING PAGES: | Click for more details
You can add pages to your website by copying and pasting any one of the pages and renaming it to whatever you need. You can then add links from the other pages or on the menu.

CHANGING THE GRAPHICS/PICTURES: | Click for more details
You can replace the page pictures with your own. The pictures are generally now 100 x 67 and 100 x 150 pixels but you can use any size picture you want and the HTML pages will re-size to fit automatically. Most pages have their own pictures which are named according to the page they're on. You will find all pictures in the "picts" folder.
